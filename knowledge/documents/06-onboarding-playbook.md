# Lightforth — Onboarding Playbook

## Overview

Most Lightforth onboarding is self-serve and fast — the goal is a tailored resume and a first Auto-Apply batch inside 15 minutes of signup. Coach/organization bundles have one extra step: seat setup.

---

## Individual Onboarding (Self-Serve)

### Minute 0-5: Sign Up
- Create account, verify email
- Upload existing resume (or start from a template)

### Minute 5-15: First Value
1. Paste a real target job description
2. Resume Builder tailors the resume and shows the ATS score
3. Review and download, or push straight into Auto-Apply

### Day 1-3: Get Rolling
- Set Auto-Apply preferences (roles, locations, must-haves)
- Review the first match list and approve applications
- Run one Interview Prep session for the most likely question type

### First Real Interview
- Download the desktop app for Interview Copilot / Coding Copilot / Meeting Copilot (Premium)
- Grant screen + audio permissions
- Do a 2-minute test run before the real call

---

## Coach / Organization Bundle Onboarding

### Day 0: Bundle Purchased
- Admin account created, seats provisioned
- Welcome email with a bulk invite-code sheet for their cohort

### Day 1: Distribute Seats
- Admin shares individual invite links (or a shared signup code) with each person in the cohort
- Each person creates their own account — their resume/application/interview content is never visible to the org admin

### Day 1-2: Admin Dashboard Walkthrough (30 min)
- Seat activation status
- Aggregate usage view (no individual content)
- Billing and seat management

### Ongoing
- Org admin can add/remove seats anytime; unused seats aren't billed until activated

---

## Common Onboarding Issues

### "I uploaded my resume but the tailored version looks off"
- Make sure the original resume has real content in each section (not just a template) — the tailoring engine works best with complete source material.

### "Auto-Apply isn't finding matches"
- Broaden role titles or locations in Auto-Apply preferences; very narrow filters return few or no matches.

### "Copilot isn't picking up audio"
- Confirm screen + audio permissions were granted to the desktop app, and that the correct audio device is selected in settings.

### "My cohort's invite code isn't working" (bundle)
- Confirm the seat was actually activated by the org admin — an invited-but-unactivated seat can't complete signup yet.
