# Lightforth — Onboarding Process (Quick Reference)

## Individual Customer

### First 15 Minutes
1. Sign up, verify email, upload existing resume
2. Paste a real target job description into Resume Builder
3. Review the tailored resume + ATS score
4. Set Auto-Apply preferences (roles, locations, must-haves)

### First Week
- Approve the first Auto-Apply match batch
- Run one Interview Prep session
- If Premium/Pro and an interview is scheduled: download the desktop app, grant screen/audio permissions, run a 2-minute test before the real call

### If They Get Stuck
- Resume looks off → check the source resume has real content, not a bare template
- No Auto-Apply matches → broaden role/location filters
- Copilot not hearing audio → confirm permissions + correct audio device selected

---

## Coach / Organization Bundle

### Day 0
- Admin account created, seats provisioned, invite sheet sent

### Day 1
- Admin distributes individual invite links/codes to the cohort
- Each person signs up on their own account — content stays private from the org admin

### Ongoing
- Admin dashboard: seat activation + aggregate usage only, no individual content visibility
- Seats reassignable anytime at no extra cost
