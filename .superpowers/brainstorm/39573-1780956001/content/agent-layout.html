One card per agent showing its current status, stats, and a progress bar. At-a-glance overview. You can click into each card to see that agent's detail log.

C — Pipeline View

Shows each job as a card moving through the 4 agent stages (Found → Filtered → Tailored → Applied). Like a Kanban board for job applications.

Click a card to select your preferred layout, or tell me in the terminal if you want a combination.
